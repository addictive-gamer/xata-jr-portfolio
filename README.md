## 🤖 Información para IA y Desarrolladores (Prompt Logic)
Este repositorio incluye notas para que otros usuarios o IAs puedan replicar la estructura:
* **Lógica de Idioma:** Uso de atributos `data-lang` en el `body` para alternar entre clases `.es` y `.en` sin recargar la página.
* **UI/UX:** Estética *Glassmorphism* con filtros de desenfoque (`backdrop-filter`) y paleta de colores neón/púrpura.
* **Prompt Sugerido:** > "Genera un portafolio estático bilingüe en HTML/JS. Crea un visor de imágenes modal con un efecto de lupa (zoom lens) que use una imagen duplicada escalada y se mueva con el cursor del mouse."
