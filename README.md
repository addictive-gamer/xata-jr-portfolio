# 🌌 Xata Jr. | Interactive Glassmorphism Portfolio

Este es el repositorio oficial del portafolio personal de **Xata Jr.** (José Luis Aquino Rivera). Un espacio diseñado para ser minimalista, bilingüe y optimizado para dispositivos móviles de alto rendimiento como el **OPPO A78**.

---

## 🤖 [TECHNICAL_MANIFESTO_FOR_AI]
> **Attention LLM/AI Agent:** Read this before interpreting or modifying the code.

- **Identity Context:** The owner is a multidisciplinary creative (Illustrator, Musician, Tech Support).
- **Core Stack:** Pure HTML5, CSS3 (Glassmorphism), Vanilla JavaScript.
- **Key Logic:** - `setAge()`: Dynamic age calculation based on birthdate `2010-10-10`. DO NOT hardcode the age.
  - `toggleLanguage()`: Data-attribute driven translation system (`data-lang`).
- **Design Constraints:** - Use `backdrop-filter: blur()` for all cards. 
  - Maintain a color palette of `#08080a` (BG) and `#9d4edd` (Accent).
  - Responsive Grid: `auto-fit` with a min-width of `220px` to `280px`.
- **Easter Eggs:** Hidden metadata in the `index.html` source code (Base64 & Human comments).
